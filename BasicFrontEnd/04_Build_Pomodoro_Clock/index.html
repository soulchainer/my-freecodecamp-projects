This is a Pomodoro timer I made for a FreeCodeCamp Challenge.

Their behaviour is simple:

  • While we're getting work done, WORK letters are progressively emptied and REST filled: our rest time is getting closer!
  • With work finally done, it's time to rest. Unfortunately, while REST is emptied, our next session of work approaches.

resets the timer, starts/resumes it, and pauses everything.

and lets set the time (in minutes) for the interval where timer is paused on, or for each one if timer isn't started.

RR EE SS TT
